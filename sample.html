Night Mode

By Ansh C

This is night mode in HTML/CSS/JavaScript. It works by adding the classname "nightmode" to an text element (p, h1, h2, and so on tags)

Use the toggle button at the end of this page to switch between night and light mode or click "Night Mode" on the top of this page.

Change the CSS Brand Color to your preferred color

With different classes, you can change different elements like this! It turns into the brand color when Night Mode is toggled. Feel free to inspect this page to view the classes used, and the CSS too!

Change the darkmain variable in the :root of the css file to change the main night mode color. By default its set to rgb(36, 40, 44). The variables include: --brandcolor | --darkmain | --darksecondary | --darktertiary | --lightgrey. Feel free to change or add any of the variables according to your needs!

This is the 'nightmode-s' class. Its called the nightmode switch class. In the light theme, it keeps the brand color, and in nightmode, it turns into white!

This is the Night Mode Div Class ('nightmode-div'). It uses the secondary dark color from the CSS and uses it as its background. It's darker than the main dark color in "nightmain"

It also provides a box shadow when in light mode!

A list of the classes used and what they do:

  • nightmain: this class is the main class which is wrapped around all nightmode elements, without this night mode will not work. As an example, you can put it on the body tag to make all of the HTML elements workable with nightmode. This also acts as the background for the night mode elements. In dark mode, the element with this class turns into ('--darkmain')
  • nightmode: This is the main nightmode class. It turns any text element into the brandcolor when in night mode, and turns any text element to black when in light mode most of the classes are built off this class
  • nightmode-w: This class works like the 'nightmode' class, except it turns elements white in night mode and black in light mode
  • nightmode-s: This class is the nightmode 'switch' class. In light mode, it turns any text element into the brand color, and in night mode, it turns to white
  • nightmode-div: This is the night mode class for divs. As seen above, this helps in styling divs for night mode. In night mode, the div's background turns into a dark color ('--darksecondary'). In light mode, the div's background turns into white and generates a box shadow to make it visible
  • nightmode-div2: This is class works like the first div class except it uses a darker background color in night mode ('--darktertiary'). In light mode, it has a grey background color ('--lightgrey'), and no box shadow
